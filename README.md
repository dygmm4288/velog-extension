# velog-preview-extension

velog 블로그 글 작성을 더 편리하게 해주는 크롬 확장 프로그램

# 기능

- Preview 없애기
- 템플릿 저장 및 불러오기
- 글쓰기 페이지에서 다크테마 버튼 사용 가능

# 설치 방법

1. git clone
2. chrome://extensions/ 접속 \* arc::/extensions/
3. 오른쪽 상단 develope mode 체크
4. "Load unpacked" 클릭
5. clone한 폴더 선택
